# DIAN Compliance Platform - Frontend

This is the frontend application for the DIAN Compliance Platform.

## Development

To start the development server:

```bash
cd frontend
npm install
npm run dev
```

## Build

To build for production:

```bash
cd frontend
npm install
npm run build
```

## Technologies Used

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Axios for API calls

## Project Structure
